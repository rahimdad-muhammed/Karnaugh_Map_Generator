# Karnaugh Map Generator - Program Description
This program will build K-map based on your boolean equation.

# How to use it?
- By pressing any letter on your keyboard you can add a input type to your equation (A,B,C,etc.)
- By pressing the SPACE key you can enable the prime mode in prime mod what ever you write you will see a upperline on your input, you can disenable it by pressing the space key again.
- By pressing to BACKSPACE key you can delete the things you just entered
- By pressing to ("^","+") symbols on your keyboard (the keys above the letters) you can add or, xor operations to your boolean equation.
- If you enter input types next to next the and operation will be used otomatically so you don't have to do anything to add this operation.
- If you press to ENTER key the Karnaugh-Map will be created based on your equation.
- If you click to CLEAR button that under the text box you can reset everything.

  # How to run it?
  - Make sure Python installed on your device.
  - After that install the pygame
  - Run the main.py file
  - Now you got running program
